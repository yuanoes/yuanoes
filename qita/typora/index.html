# Typora 简介

Typora 是一款简洁的 markdown 编辑器，没有采用源代码和预览双栏显示的方式，而是采用所见即所得的编辑方式，实现了即时预览的功能。

# 自定义 typora 主题

  1. 可以先去 typora.theme.io 选取合适的主题或者直接使用自带主题

  2. 使用方式打开偏好设置打开外观界面选择打开主题文件夹

  3. 把下载好主题的 css 文件解压到主题文件夹里

  4. 打开设置中的调试模式，右键检查元素，可以看到 typora 的本质实际上就是一个浏览器

  6. 我们就可以先在浏览器里修改调试 css 样式再到原主题文件里修改
