# 前言

一俩月没更过博客了，可能以后也会很久才更新一篇了，因为翻了翻这学期我写的博客，不说和网上一摸一样（太水了），只能说只能当一个方便我查看笔记的地方，所以以后博客只会写一下个人总结和 ctfwp 类似自己纯手打的东西了，而学习笔记放到 notion 里组建一个 web 安全知识库，（notion 是一个 word+markdown+tpora + 博客一样的宝藏软件）其实我发现自己写的博客也会有人看的，之前博客接入过百度统计，有一些 seo 优化，看到每天的 ip 数量增加，还是会很高兴的，之前看 b 站 python 教程，在评论下分享了下那篇速成文章，然后收获挺多的点赞感谢 hhh。

7 月考完最后一门英语，我的大二也彻底结束了，现在暑假也差不多过去了一半，若有所失。

# 总结

大概也是从大二下学期开始正式接触安全的吧，其实最初接触安全是从 0xgame 开始的，因为 web 比较好入门，就选择了 webdog 这条路，其实刚开始学习 web 真的挺懵的也不知道要干啥非常迷茫，直到了解一些真心觉得 web 安全入门真的难，虽然 ctfer 公认的 web 入门比较简单，而 web 手一般会搭配 misc 或者密码学来学，我只能说人与人的体质不能一概而论。

其实这个假期想去找个实习的，但因为时间问题也没去了，明年寒暑假再找了，大三再找实习应该也不算晚。找实习的时候写了一个简历，写了很久，因为每当我想要写简历的时候，我就在想，我参加 ctf 获得了什么奖了吗？没有。有什么护网经历吗？没有。有实战经历有漏洞证书吗，没有。有出过 ctf 的题目吗？，没有，有加入什么厉害的 ctf 战队吗？没有，有自己做过什么厉害的项目吗？学生管理系统算嘛自己会什么？似乎什么都不会。
